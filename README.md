# Pac-Man Game

A JavaFX-based Pac-Man game developed for the Programming II course project.

## Features

- User Registration and Login
- Multi-Level Gameplay
- Ghost Enemy System
- Collision Detection
- Pellet Collection
- Score Tracking
- High Score Management
- File-Based Persistence
- Game Over and Level Progression

## Technologies Used

- Java 17
- JavaFX
- FXML
- Object-Oriented Programming (OOP)
- File I/O Operations

## Project Structure

src/
├── controller/
├── model/
├── utils/
├── game.fxml
├── menu.fxml

## How to Run

1. Open the project in IntelliJ IDEA.
2. Configure JavaFX libraries.
3. Run Main.java.
4. Register or log in.
5. Start playing.
